# Project Auth

## Google Auth
Use https://console.cloud.google.com/projectcreate to add Google Config

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation
Node v20.11.0, npm 10.2.4

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
